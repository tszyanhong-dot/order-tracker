使用教學（免費建立專屬雲端）

  1. 去 Supabase 官網 免費註冊一個帳號。
  2. 建立一個新 Project (名稱隨便，密碼自己記住，Region 選 Tokyo 東京速度最快)。
  3. 在專案頁面，搵到左邊選單嘅 SQL Editor，點擊 New query。
  4. 將下面嘅 SQL 碼複製貼入去，然後點擊防右下角嘅 Run： 
    create table products (
  id bigint primary key,
  name text not null,
  photo text
);

create table orders (
  id bigint primary key,
  customer text not null,
  productid text,
  productname text,
  productphoto text,
  variant text,
  qty integer not null,
  jpyprice numeric,
  hkdprice numeric,
  status text default '未採購',
  time text
);

alter table products disable row level security;
alter table orders disable row level security;
  5. 去左下角 Project Settings -> API，複製 Project URL 同 anon / public 金鑰貼到上面儲存。
  6. 喺你其他裝置（iPad / 手提電腦）打開同一個網址，輸入相同 URL 同 Key 即可即時同步！
